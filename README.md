## Descripción del Proyecto

Sabores del Chef es una escuela culinaria que ofrece cursos profesionales y recreativos de gastronomía. Imparte clases de cocina colombiana, internacional, repostería, panadería y coctelería. Cuenta con 12 chefs instructores, 5 cocinas equipadas y ofrece certificaciones profesionales. Atiende 150 estudiantes mensualmente.

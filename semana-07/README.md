# Semana 07

## Descripción

Organización del código en paquetes.

## Estructura de Archivos

```
src/
└── com/
    └── proyecto/
        ├── Main.java
        ├── modelo/
        ├── servicio/
        └── excepciones/
```

## Instrucciones de Compilación y Ejecución

### Compilación

```bash
javac -d . src/com/proyecto/**/*.java
```

### Ejecución

```bash
java com.proyecto.Main
```
